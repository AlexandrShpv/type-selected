Translation Fix Test

✅ Translation Issue Fixed:

Problem: Translation was showing fallback responses like "[RU] text" instead of actual translations.

Solution: Removed unreliable CORS proxy and added direct Google Translate API call with improved fallback dictionary for basic translations.

How to Test:

  1. Open Chrome Developer Tools (F12)
  2. Go to Console tab
  3. Copy and paste the content of type-selected.js
  4. Select any text below
  5. Check translation for each language option
  6. Should show actual translations, not "[EN] text"
